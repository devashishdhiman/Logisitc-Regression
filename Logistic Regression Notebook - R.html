Introduction to Logisitic Regression¶

Like many other machine learning techniques, it is borrowed from the field of statistics and despite its name, it is not an algorithm for regression problems, where you want to predict a continuous outcome. Instead, Logistic Regression is the go-to method for binary classification. It gives you a discrete binary outcome between 0 and 1. To say it in simpler words, it’s outcome is either one thing or another.

Logistic can be categorised as;

  • Linear Algorithm
  • Classification Algorithm
  • Supervised machine learning algorithm
  • Parameteric Algorithm

Why to use Logisitc Regression?¶

In a previous video we understood how Linear regression works and how we can use it to predict continous Y variable.

Youtube : Linear Regression

In linear regression the Y variable is always a continuous variable. If suppose, the Y variable was categorical, you cannot use linear regression model it.

So what would you do when the Y is a categorical variable with 2 classes?

Logistic regression can be used to model and solve such problems, also called as binary classification problems.

Let's take an example of student test score to understand this better.

  • Linear Regression could help us predict the student’s test score on a scale of 0 - 100. Linear regression predictions are continuous.

  • Logistic Regression could help use predict whether the student passed or failed. Logistic regression predictions are discrete.

How Does it work?¶

Logistic Regression measures the relationship between the dependent variable (our label, what we want to predict) and the one or more independent variables (our features), by estimating probabilities using it’s underlying logistic function.

These probabilities must then be transformed into binary values in order to actually make a prediction. This is the task of the logistic function, also called the sigmoid function. The Sigmoid-Function is an S-shaped curve that can take any real-valued number and map it into a value between the range of 0 and 1, but never exactly at those limits. This values between 0 and 1 will then be transformed into either 0 or 1 using a threshold classifier.

The picture below illustrates the steps that logistic regression goes through to give you your desired output.

Picture Credit : https://machinelearning-blog.com/

We want to maximize the likelihood that a random data point gets classified correctly, which is called Maximum Likelihood Estimation. You can maximize the likelihood using an optimization algorithm like Newton’s Method or Gradient Descent.

Advantages & Disadvantages¶

Advantages¶

  • It is extremely efficient
  • Does not require much resources
  • Very fast training and prediction times
  • Easy to interpret
  • Does not require inputs to be scalled
  • Easy to regularise

Disadvantages¶

  • It can only be used for linear problems
  • You would have to supervise the input variables and remove the unrelated independent variables
  • Can only predict a categorical outcome
  • Tends to overfit

What can it be used for?¶

You might wonder what kind of problems you can use logistic regression for. It is the most widly used Algorithm of all.

Here are some examples of binary classification problems:

  1. Spam Detection : Predicting if an email is Spam or not
  2. Banking : Predicting if a customer will default on a loan.
  3. Marketing : Predicting if a given user will buy an insurance product or not
  4. Credit Card Fraud : Predicting if a given credit card transaction is fraud or not
  5. Health : Predicting if a patient has diabetes or not
  6. Survival : Predicting if a passenger would survive the titanic (sort of) event
